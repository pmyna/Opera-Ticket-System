1) Ladet euch am besten den ganzen Ordner "Datenbanken" runter

2) Öffnet in Pycharm den gespeicherten Ordner als working directory 

	Im Python Terminal:
		> pip install flask
		> import flask

3) Auf dem Rechner-Terminal:
	> cd /EUER PATHWAY zum Ordner
	> export FLASK_APP=webapp.py (# damit flask weiß welches Skript er ausführen soll)
	> export FLASK_DEBUG=1 (# damit könnt ihr die Seite mit "refresh" neu laden, statt jedes mal neu starten zu müssen)
	> flask run (# script ausführen)

4) Explorer eurer Wahl öffnen:
	localhost:5000 in die Adressleiste eingeben

5) Enjoy Homepage :)
